# Imagen Open Graph (OG Image)

## ¿Qué es?

La imagen Open Graph es la imagen que aparece cuando compartes un enlace de tu sitio en redes sociales (Facebook, Twitter, WhatsApp, LinkedIn, etc.).

## Ubicación

Debes colocar la imagen en:
```
portfolio-react/public/og-image.jpg
```

## Dimensiones Recomendadas

- **Tamaño ideal**: 1200 x 630px (ratio 1.91:1)
- **Formato**: JPG o PNG
- **Peso**: Menos de 1 MB (idealmente 200-500 KB)

## Contenido Sugerido

La imagen debe incluir:
- Tu nombre o logo
- Título: "Fabián - Portafolio"
- Subtítulo: "Diseñador Gráfico & Desarrollador Web"
- Diseño atractivo con tu estilo personal
- Colores que representen tu marca

## Ejemplo de Diseño

## Herramientas para Crear

1. **Canva** - https://www.canva.com/
   - Plantilla: "Facebook Post" (1200x630)
   - Fácil de usar, muchas plantillas

2. **Figma** - https://www.figma.com/
   - Profesional y gratuito
   - Control total del diseño

3. **Photoshop/Illustrator**
   - Si ya los usas

## Cómo Probar

Después de subir tu imagen, prueba cómo se ve:

1. **Facebook Debugger**: https://developers.facebook.com/tools/debug/
2. **Twitter Card Validator**: https://cards-dev.twitter.com/validator
3. **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/

## Notas Importantes

- La imagen se actualiza dinámicamente para cada proyecto usando la imagen principal del proyecto
- Para la página de inicio y lista de proyectos, se usa `/og-image.jpg`
- Asegúrate de que el texto sea legible incluso en tamaños pequeños
- Evita texto muy pequeño (mínimo 40px de altura)
